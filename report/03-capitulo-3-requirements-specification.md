## 3.1. User Stories.

**Épicas**
| Epic ID | Título | Descripción | Relacionado con|
| :--- | :--- | :--- | :--- |
| **EP01** | Información del Producto y Landing Page | Como visitante, quiero conocer la propuesta de valor de SpotTrack y sus funcionalidades clave (IoT Edge, mapas de calor) para comprender cómo puede optimizar la gestión de gimnasios. | N/A |
| **EP02** | Autenticación y Gestión de Perfiles | Como usuario (administrador o cliente), quiero registrarme, iniciar sesión y gestionar mi información personal para acceder de forma segura a mi panel. | N/A |
| **EP03** | Mapa de Calor y Disponibilidad | Como cliente frecuente, quiero visualizar la disponibilidad de las máquinas en tiempo real para esquivar las aglomeraciones. | N/A |
| **EP04** | Sugerencias y Reservas | Como cliente, quiero recibir recomendaciones de ejercicios alternativos o realizar reservas exprés en horas pico para no perder el "pump". | N/A |
| **EP05** | Dashboard y Estadísticas de Uso | Como administrador, quiero ver gráficos sobre el uso histórico y afluencia de máquinas para comprender la demanda real. | N/A |
| **EP06** | Mantenimiento Predictivo y Tickets | Como gerente de operaciones, quiero recibir alertas automáticas cuando un equipo supere sus horas seguras de uso. | N/A |
| **EP07** | Gestión de Activos y Multisede | Como administrador de una cadena, quiero gestionar el inventario, repuestos y cambiar la vista entre locales. | N/A |
| **EP08** | Analítica Financiera e Impacto | Como dueño del negocio, quiero calcular el impacto financiero por inactividad y analizar el ROI para tomar decisiones rentables. | N/A |
